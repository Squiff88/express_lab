------CREATE STORED PROCEDURE ANDRETURN A RANDOM ROW---------- CREATE PROCEDURE upToTwentyFivev AS BEGIN SELECT TOP 1 PP.FirstName , PP.LastName,HE.BirthDate , HE.Gender FROM [HumanResources].[Employee] HE INNER JOIN [Person].[Person] PP ON PP.BusinessEntityID = HE.BusinessEntityID WHERE HE.BirthDate > '1975' and HE.BirthDate < '1980' ORDER BY NEWID() END ------- RUN THE STORE PROCEDURE ------------ EXEC upToTwentyFive
